## これはどういうことですか？

🐞 私たちのプロジェクトはますます複雑化しており、単一ブランドのツールでは対応しきれないことが多くなっています。オープン BIM はこの問題を解決するもので、私たちのワークフローの中でますます存在感を増しており、**IFC はその基盤となっています**。

<IfcCard>
  IFCは、異なるベンダーのアプリケーション間で情報を交換するだけでなく、単一のアプリケーション/ライセンスポリシーに依存せずに情報にアクセスすることを可能にします。
</IfcCard>

🌈 しかし、ドキュメンテーションは簡単ではなく、このスキームから本当に重要なことを見極めるのは難しい場合が多いのです。**このコースでは、IFC.js を開発するために必要な知識をすべてお教えします**。

🌟 **このコースはプログラミングコースではありません**。その代わりに、IFC スキーマがどのように機能し、どのような情報を含み、どのように構造化されているかを理解することを目的としています。
