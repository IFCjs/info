### 何にお金がかかるの？

👌
**絶対に何もない**、時間もない。簡単に言えば、BIM コードのすべてを自分のために確保する代わりに、その一部を IFC.js にオープンにしておくことができるのです。

<IfcAlert>どのリポジトリにコードを保存するかだけの問題です。</IfcAlert>
