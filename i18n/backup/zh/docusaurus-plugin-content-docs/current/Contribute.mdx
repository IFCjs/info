🔥
当然，你可以在你的闭源程序中实现这些功能......或者你可以\*\*直接在 IFC.js 中实现它们。

### 我花了多少钱？

👌
**绝对没有**，甚至没有时间。简单地说，与其把所有的 BIM 代码保留给自己，不如在 IFC.js 中留下一部分代码。

<IfcAlert>这只是一个问题，你把你的代码保存到哪个仓库。</IfcAlert>
