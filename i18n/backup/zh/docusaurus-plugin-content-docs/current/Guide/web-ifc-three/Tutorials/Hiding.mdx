## 如何做到这一点

### 获取类别的子集

👩‍🏫
IFC.js 的可见性是基于**子集操作的。这使得复杂的可视化能够以最小的内存占用来创建。我们要做的第一件事是定义哪些**IFC 类别\*\*我们将允许用户显示或隐藏。

📋📋📋
为了节省内存，IFC.js 中的**类别被定义为数字常数**。因此，让我们创建一个对象，将这些常量的名称映射到它们的数字值，并创建一个函数来检索它们。

```js
import {
  IFCWALLSTANDARDCASE,
  IFCSLAB,
  IFCDOOR,
  IFCWINDOW,
  IFCFURNISHINGELEMENT,
  IFCMEMBER,
  IFCPLATE,
} from "web-ifc";

// 类别名称列表
const categories = {
  IFCWALLSTANDARDCASE,
  IFCSLAB,
  IFCFURNISHINGELEMENT,
  IFCDOOR,
  IFCWINDOW,
  IFCPLATE,
  IFCMEMBER,
};

// 获取一个类别的名称
function getName(category) {
  const names = Object.keys(categories);
  return names.find((name) => categories[name] === category);
}
```

🏠⏩🚪
现在让我们创建几个函数，以**获得属于一个类别的元素的所有 ID**，并**用这些 ID 创建一个子集**。

<IfcAlert>记住，许多IFC.js函数是异步的，所以我们要使用async和await。</IfcAlert>

👩‍🏫
你也可以使用`removeFromSubset()`从一个子集中删除一个项目（例如，隐藏一个项目）。如果你把它和`createSubset()`结合起来，并且`removePrevious = false`，你就可以完全控制哪些内容被添加到哪个子集以及它的可见性。

```js
// 获取一个特定类别的所有项目的ID
async function getAll(category) {
  const manager = ifcLoader.ifcManager;
  return manager.getAllItemsOfType(0, category, false);
}

// 创建一个新的子集，包含一个类别的所有元素
async function newSubsetOfType(category) {
  const ids = await getAll(category);
  return ifcLoader.ifcManager.createSubset({
    modelID: 0,
    scene,
    ids,
    removePrevious: true,
    customID: category.toString(),
  });
}
```

### 将 GUI 与可见性结合起来

✨
现在只剩下**将每个复选框的点击事件与相应类别的可见性联系起来**，这样，当复选框被激活时，类别就变得可见，而当它被停用时，类别就消失了。

🎨👨‍🎨
这里值得注意的是，为了使代码更加简洁，我们给**每个复选框一个 ID，其名称与它所链接的类别相同**。

```js
// 存储已创建的子集
const subsets = {};

async function setupAllCategories() {
  const allCategories = Object.values(categories);
  for (let i = 0; i < allCategories.length; i++) {
    const category = allCategories[i];
    await setupCategory(category);
  }
}

// 创建一个新的子集并配置复选框
async function setupCategory(category) {
  subsets[category] = await newSubsetOfType(category);
  setupCheckBox(category);
}

// 设置复选框事件来隐藏/显示元素
function setupCheckBox(category) {
  const name = getName(category);
  const checkBox = document.getElementById(name);
  checkBox.addEventListener("change", (event) => {
    const checked = event.target.checked;
    const subset = subsets[category];
    if (checked) scene.add(subset);
    else subset.removeFromParent();
  });
}
```
